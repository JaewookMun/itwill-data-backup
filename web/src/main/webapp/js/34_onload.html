DOM(Document Object Model) - onload 이벤트

이벤트(Event) : 태그(박스모델 - Element 객체)에 관련된 사건

태그에서 이벤트가 발생될 경우 실행될 명령을 자바스크립트로 작성하여 이벤트 처리

동적인 페이지를 구현하기 위해서는 DOM Tree가 완성된 후 DOM Tree의 엘리먼트 객체를 조작하여 구현

DOM Tree : document 객체를 기반으로 태그(박스모델)를 엘리먼트 객체로 표현하여 트리(Tree) 형식의 계층적 구조로 표현한 형태

DOM Tree는 웹문서의 태그를 모두 읽은 후(onload) 완성 - DOM Tree가 완성된 후 조작 가능
